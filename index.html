Welcome to CrimsonCTRL

This project is part of the CrimsonUniverse which contains:

  • CrimsonCore

A SparkCore Microcontroller (Spark.io). This is the master control unit of the project, attached to the CrimsonCore is a photo resistor to measure the brightness. If nights fall a relay is switched on and the light of the StoneCircle is switched on. Furthermore a DS18B20 is measuring the temperature. Last not least a Digole 1,8" OLED display is used to visualise the data.

  • CrimsonControl

Is a HTML5 frontend that receives the data from the SparkCloud.

  • CrimsonCAM

Is a camera module attached to a Raspberry PI. The software motion is used for motion-tracking.

  • CrimsonCTRL

This project is a native Android application. You can manual switch the Stone Cirle Light on or off and get the temperature and the values from the photo resistor
